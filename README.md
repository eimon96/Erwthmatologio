Οι διαφημίσεις προσθέθηκαν για εκπαιδευτικούς σκοπούς.

- Uncomment τα σημεία του κώδικα που τις αφορούν 

- Add τα credentials από το AdMob

- Sync project with gradle

- Build

- BOOM διαφημίσεις
